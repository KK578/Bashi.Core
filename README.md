Bashi.Core is a collection of assorted domain-agnostic utility methods that become useful to share across multiple projects.

## Building from Source

Requirements:
 - .NET Core SDK 3.1.x
 
```sh
$ git clone https://github.com/KK578/Bashi.Core.git
$ cd Bashi.Core
$ dotnet build
$ dotnet test
```
